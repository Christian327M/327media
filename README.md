# 327 Media

**Live site:** [https://327media.org](https://327media.org)

Your one-stop dashboard for **markets, politics, sports, betting, and the CIA Podcast**.

### What is 327 Media?
- Real-time stock ticker + TradingView charts & heatmap  
- Auto-updating news from Reuters, AP, Fox, CNN, WIRED, etc.  
- White House live links + President Trump updates  
- Left & Right media coverage  
- Sports, prediction markets & betting hub  
- Featured CIA Podcast episodes  
- Full login/signup with Supabase  

### Tech Stack
- Single-file HTML + Tailwind CSS (GitHub Pages)  
- Supabase Auth  
- TradingView widgets  
- FormSubmit for contact & writer applications  
- RSS feeds via allorigins.win

### How to contribute
Want to write articles or commentary? Go to the **“Write for Me”** section on the live site!
